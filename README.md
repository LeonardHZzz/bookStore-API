# bookStore-API

Funcionalidades Clave

Catálogo Dinámico: Gestión completa (CRUD) de libros, incluyendo metadatos como ISBN, autores y editoriales.

Búsqueda Avanzada: Filtros por categoría, precio, fecha de publicación y popularidad.

Gestión de Inventario: Control de existencias en tiempo real y alertas de bajo stock.
